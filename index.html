How to Use This Website

  1. Select the root folder that contains your topic subfolders (e.g., Physics, Chemistry).
  2. Choose which topics you want to include using the checklist.
  3. Set the number of questions per topic and exam duration.
  4. Start the test — questions will appear one at a time with navigation.
  5. Use the sidebar to jump between questions, and mark questions for review.
  6. At the end, submit to download a PDF of your responses.
