# CSIR-UGC NET Topic-wise Mock Test Website

This project is a customizable mock test platform designed for CSIR-UGC NET Physics preparation. It allows users to select topic folders, choose which subjects to include, and set default or per-topic question counts. The platform features a timer, sidebar navigation, mark-for-review options, and PDF export of responses with embedded images and topic labels. A built-in Help menu provides clear usage instructions and links to downloadable CSIR NET Physics topic-wise folders from Google Drive:

- https://drive.google.com/drive/folders/1Di8MUlFYWjfKyMK_IcbuglPotpUDeYGt?usp=drive_link  
- https://drive.google.com/drive/folders/1i2LzDWPRwi0MzJWp5Qt4vrnLh4WspdMH?usp=drive_link  

This makes the mock test highly flexible, exam-like, and easy to use for learners preparing for CSIR NET Physics.
